# 🎵 Gift of Music Foundation – Charity Website

A modern, premium, responsive multi-page charity website built using **HTML, CSS, and vanilla JavaScript**.  
Designed with a **dark, futuristic SaaS-inspired UI** to demonstrate high-end frontend development skills.

---

---

## 📌 Project Overview

This project is a fictional charity website for **Gift of Music Foundation**, an organization focused on using music as a tool for:

- Healing
- Education
- Community empowerment
- Social impact

The website was designed to match a **premium SaaS-style aesthetic inspired by modern platforms like Starlink**, focusing on clean layouts, strong typography, and minimal yet impactful UI.

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript (for interactivity)
- Responsive Web Design

---

## 🎨 Design Highlights

- Dark, premium futuristic UI
- Large hero sections with strong typography hierarchy
- Minimalist SaaS-style layout
- Electric blue accent system
- Glassmorphism-inspired components
- Smooth hover transitions and UI feedback
- Fully responsive mobile-first design

---

## 📄 Pages Included

- **Home** – Hero section, impact overview, and event previews  
- **About** – Mission, story, and values  
- **Services** – Music education, outreach, and therapy programs  
- **Events** – Upcoming events displayed in card layout  
- **Donate** – Interactive donation UI with selectable amounts  
- **Contact** – Contact form and organization details  

---

## ⚙️ Features

- Multi-page navigation system
- Sticky responsive navbar
- Mobile hamburger menu (CSS + JavaScript)
- Interactive donation selection system
- Custom amount input handling
- Button state animations and hover effects
- Fully responsive across devices

---

## 💡 Key Learning Outcomes

This project demonstrates:

- Structuring multi-page websites without frameworks
- Building responsive layouts using Flexbox and Grid
- Creating interactive UI with vanilla JavaScript
- Designing premium SaaS-style interfaces
- Handling real-world UI/UX constraints

---

## 📱 Responsive Design

The website is fully optimized for:
- Desktop
- Tablet
- Mobile devices

Navigation collapses into a mobile-friendly hamburger menu for smaller screens.
